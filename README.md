# node-excel-json

> 有时候需要把excel的内容转成需要的json格式，以前手动的复制粘贴实在是麻烦。。
    所以在网上找了下node-xlsx的案列，然后仿照着自己改了下json的格式就好啦
    
### 安装：
``` 
npm install
npm install node-xlsx
```
### 执行：
```
node index.js
```
